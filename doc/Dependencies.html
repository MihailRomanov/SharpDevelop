SharpDevelop can take advantage of the following software if you install it:

  • Microsoft .NET Framework 4.7.2 Developer Pack or later for .NET code completion documentation.
  • Latest Microsoft Windows SDK (optional).
  • Microsoft Build Tools for common MSBuild targets support
    • Microsoft Build Tools 2015
    • Visual Studio 2017 Build Tools
    • Visual Studio 2019 Build Tools
  • TortoiseGit for Git source control support.
  • TortoiseSVN for Subversion source control support.
  • SHFB for documentation generation support.
  • WiX Toolset 3.X for building installers.
